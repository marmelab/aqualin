# Aqualin Console

Play the game of Aqualin on the console ([game presentation](https://www.kosmosgames.co.uk/games/aqualin/), [rules](https://www.ultraboardgames.com/aqualin/game-rules.php)).

## Requirements

- Node.js

## Installation

```sh
yarn
```

### Playing

```sh
yarn start
```

### Developing

```sh
## Re-run the script on change
yarn dev

## typecheck && compile
yarn build

## format
yarn prettier --write .

## test
yarn test
```
